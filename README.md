# Seyedmahdiar Ziaolhagh — Portfolio V8

## Preview locally
Open the folder in Visual Studio Code and run **Live Server** from `index.html`.

## Deploy
The site is static and can be deployed to GitHub Pages, Netlify, Cloudflare Pages, or any ordinary web host. Upload the **contents** of this folder so that `index.html` is at the site root.

## V8 refinements
- Shared accessibility layer and keyboard-visible focus states
- Skip-to-content links and reduced-motion support
- Responsive image dimensions, lazy loading, and WebP assets
- Social-sharing preview, favicon, Apple touch icon, and page metadata
- Print-friendly styles and custom 404 page
- Safer external-link behavior and video metadata loading

## Before adding a custom domain
Add absolute canonical URLs and generate a domain-specific sitemap only after the final domain is known.
